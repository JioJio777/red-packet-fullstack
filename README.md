# 红包系统 · 项目立项文档

## 项目概述

基于 Go + React 的全栈红包系统，支持用户发送、领取红包等核心功能。

这是一个练习系统，目的不是为了上线，而是为了练手，目前只是本地运行

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React |
| 后端 | Go |
| 通信 | RESTful API |
| 数据库 | MySQL |
| 缓存 | Redis |
| 认证 | JWT |

## 核心功能（初步规划）

- 用户注册 / 登录
- 发红包（普通红包、拼手气红包）
- 领红包
- 红包记录查询
- 账户余额管理

## 项目结构（初步规划）

```
fullstackProject/
├── backend/          # Go 后端
└── frontend/         # React 前端
```
